# Анализ контента Netflix

**Проект по исследованию данных** о фильмах и сериалах, доступных на Netflix по состоянию на 2021 год.

## 📊 О проекте
Этот проект создан в рамках моего портфолио для демонстрации навыков анализа и визуализации данных. Я провел полный цикл исследования: от загрузки и очистки данных до построения графиков и формулирования выводов.

**Ключевые вопросы, на которые я искал ответы:**
*   Как менялось количество выпускаемого контента по годам?
*   Какие страны производят больше всего контента для Netflix?
*   Каково соотношение фильмов и сериалов?
*   Кто из режиссеров представлен на платформе чаще всего?

## 🛠️ Используемые инструменты
*   **Язык:** `Python 3`
*   **Библиотеки:** `pandas` (обработка данных), `matplotlib` и `seaborn` (визуализация)
*   **Среда разработки:** `Jupyter Notebook`

## 📈 Краткий обзор ключевых выводов

1.  **Фильмы против Сериалов:** Фильмы явно доминируют, составляя около **69%** контента, в то время как на сериалы приходится только **31%**.
2.  **Годы выпуска:** Большая часть контента была выпущена после 2015 года, с пиком в **2018–2020** годах. Это хорошо видно на гистограмме распределения.
3.  **Страны-лидеры:** **Соединенные Штаты** производят подавляющее большинство контента для Netflix, за ними следуют **Индия** и **Великобритания**.
4.  **Топ режиссеров:** В датасете чаще всего встречаются такие режиссеры, как **Jan Suter**, **Raúl Campos** и другие, которые специализируются на создании фильмов для разных аудиторий.
